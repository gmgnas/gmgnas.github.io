Bienvenido al Proyecto de RStudio

Este es el repositorio de mi proyecto de análisis de datos y visualización con R.

Descripción del Proyecto

Aquí puedes proporcionar una descripción más detallada de tu proyecto. Explica los objetivos, las preguntas que intentas responder o los problemas que buscas resolver.

  • Utilizamos R para el análisis estadístico.
  • Generamos visualizaciones informativas con ggplot2.
  • El código fuente se encuentra en el directorio R/.

Contenido del Repositorio

  • R/: Contiene todos los scripts de R utilizados en el proyecto.
  • data/: Almacena los conjuntos de datos necesarios para el análisis.
  • doc/: Incluye documentación adicional, como informes o archivos PDF.
  • otros/: Para cualquier otro archivo relevante.

Cómo Empezar

  1. Clona el repositorio a tu máquina local: 
    git clone https://github.com/sindresorhus/del
  2. Abre el archivo .Rproj en RStudio.
  3. Ejecuta los scripts en el directorio R/ para replicar los análisis.

Dependencias

Este proyecto utiliza las siguientes librerías de R:

  • dplyr
  • ggplot2
  • tidyverse

Puedes instalarlas con el siguiente comando en la consola de R:

install.packages(c("dplyr", "ggplot2", "tidyverse"))
